# Changelog

All notable changes to the Antigravity Remote Connect extension will be documented in this file.

## [0.1.0] - 2026-06-23

### Added
- 🏠 Secured Local Connect — LAN-only HTTPS with QR code
- 🌍 Secured Remote Connect — ngrok tunnel with HMAC-signed magic links
- 📱 Mobile-optimized web app with dark glassmorphism UI
- 🔒 Zero hardcoded secrets — all tokens generated at runtime
- 📡 Real-time session mirroring via Chrome DevTools Protocol
- ⚡ Remote control — send messages, stop generation, approve actions
- 🔄 AI model and mode switching from mobile
- 📜 Chat history browsing
- 🔐 Per-IP rate limiting and single-use nonce tokens
- 💓 WebSocket heartbeat with auto-reconnect
- 🎨 Premium mobile UI with glassmorphism and micro-animations
